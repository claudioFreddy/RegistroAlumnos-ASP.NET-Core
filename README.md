# Registro de Alumnos - ASP.NET Core MVC

Sistema académico para administrar:
- Alumnos
- Cursos
- Inscripciones
- Notas por curso

## Tecnologías
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Razor Views

## Funcionalidades
✅ CRUD Alumnos  
✅ CRUD Cursos  
✅ Inscripción de alumnos  
✅ Asignación y edición de notas  
...
